COMP 125 - Assignment 1: Interactive Mad Libs-style Story Generator 🎭

Instructions ✍️

In this assignment, you’ll be creating an interactive Mad Libs-style story generator. Your task is to animate the following scenario:

“Once upon a time, there was a student at Centennial College named (Name). One day, while studying (Subject), a (Animal) ran into the classroom. The professor, who was wearing a (Color) tie, jumped onto his (Furniture) in surprise. The whole class, including (Name), couldn’t stop (Verb ending in -ing).”

The submission deadline is 📅 Thursday, June 13 at 11:59pm. Late submissions will follow the late submission policy (max 5 days and max 50%), so ensure you submit on time.

Key Concepts You’ll Need to Know 📚

This assignment focuses on the following JavaScript concepts:

  • Input Retrieval: Retrieve user input from the provided fields in the HTML. Ensure that each field has an input; otherwise, alert the user with a message: “You didn’t enter anything. Try again.”
  • String Concatenation: Assemble the inputs into a coherent story.
  • Control Structures: Use if statements to validate the user inputs and ensure none are left blank.
  • DOM Manipulation: Display the completed story in the HTML. Additionally, change the body background color to the color set for the tie in the story.
  • Local Storage: Learn and implement local storage to save and retrieve the latest user’s story inputs, even if the page is refreshed.
  • Event Handling: Handle click events with JavaScript and alert the user if any input field is left empty.
